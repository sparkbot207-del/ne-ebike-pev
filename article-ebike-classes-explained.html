E-Bike Classes Explained

Class 1, 2, 3 — what's the difference and which do you need?

The Federal Framework

In the United States, e-bikes are classified into three classes. This system was created to standardize regulations and give states a consistent framework. Most states have adopted it, though implementations vary.

The Three Classes

Class 1: Pedal-Assist Only

  • Motor: Up to 750 watts
  • Assist stops at: 20 mph
  • Throttle: No
  • How it works: Motor only helps when you're pedaling

Where it's allowed: Most bike lanes and paths, most trails that allow bikes. Fewest restrictions overall.

Best for: Bike path users, fitness-oriented riders, trail access (where e-bikes are allowed)

Class 2: Throttle-Equipped

  • Motor: Up to 750 watts
  • Assist stops at: 20 mph (both throttle and pedal assist)
  • Throttle: Yes
  • How it works: Throttle can power the bike without pedaling; pedal assist also works

Where it's allowed: Most bike lanes and roads, some trails and paths (varies by jurisdiction). More restrictions than Class 1.

Best for: Stop-and-go commuting, riders with knee issues, delivery riders, anyone who wants throttle option

Class 3: Speed Pedelec

  • Motor: Up to 750 watts
  • Assist stops at: 28 mph
  • Throttle: Sometimes (if present, usually limited to 20 mph)
  • How it works: Pedal assist to higher speed

Where it's allowed: Bike lanes and roads (usually). Often prohibited on multi-use paths and trails.

Helmet requirement: Many states require helmets for Class 3 (minimum age may also apply)

Best for: Longer commutes, keeping up with traffic, experienced cyclists wanting speed

Why It Matters

Path and Trail Access

Land managers use classes to regulate access. A bike path might allow Class 1 only, or Class 1 and 2, but prohibit Class 3. Check local rules.

Insurance and Liability

Some jurisdictions treat Class 3 differently for registration, insurance, or age requirements.

Resale Value

Class 1 bikes have broader legal access, which can affect resale appeal.

New England State Rules

(Check current state laws — these can change)

Maine

  • Follows the three-class system
  • Class 1 and 2: Allowed where bikes are allowed
  • Class 3: Allowed on roads, may be restricted on paths
  • No registration required

New Hampshire

  • Three-class system adopted
  • E-bikes treated as bicycles (not motor vehicles)
  • Trail access varies by land manager

Vermont

  • Three-class system adopted
  • E-bikes allowed on roads
  • Path/trail access determined locally

Massachusetts

  • Follows federal classification
  • Class 3 may require registration in some contexts
  • Check municipal rules (Boston, Cambridge have specific regulations)

Rhode Island

  • Three-class system
  • E-bikes treated as bicycles
  • No registration required for classes 1-3

Connecticut

  • Three-class system adopted
  • Class 3 requires helmet
  • Minimum age may apply for Class 3

What If Your Bike Doesn't Fit the Classes?

Over 750W

Not a legal e-bike under federal framework. May be classified as moped, motorcycle, or motor vehicle depending on state.

Over 28 mph

Not a legal e-bike. These are typically regulated as motor vehicles.

Unlabeled

Manufacturers should label e-bikes with their class. If unlabeled, assume the most restrictive interpretation if you're unsure.

Bikes That Change Classes

Some e-bikes can be configured to operate as different classes:

  • Throttle can be enabled/disabled (Class 2 ↔ Class 1)
  • Speed limits can be adjusted (Class 3 ↔ Class 1)
Be honest about what mode you're using. Riding a de-restricted Class 3 on a Class 1-only trail puts trail access at risk for everyone.

The Case for Each Class

Choose Class 1 If:

  • You want maximum trail/path access
  • You prefer a workout while riding
  • You ride in areas with strict e-bike rules
  • You don't need/want a throttle

Choose Class 2 If:

  • You like the option of throttle-only power
  • You have stop-and-go commuting
  • You have physical limitations that occasionally need a break from pedaling
  • Your riding areas allow Class 2

Choose Class 3 If:

  • You have a longer commute
  • You need to keep up with traffic (roads with 30-35 mph limits)
  • You're on roads, not paths
  • You're a confident cyclist

Common Questions

Can I ride my Class 3 bike slower to use Class 1 paths?

Technically your bike's classification doesn't change based on speed. However, enforcement is rare and intent matters. Some areas allow Class 3 if limited to 20 mph.

Is 750W a lot?

It's enough for hills and headwinds for most riders. The 750W limit applies to continuous rated power — peak power during acceleration can be higher.

Are higher-class bikes "better"?

No. They serve different purposes. A Class 1 bike isn't inferior to a Class 3 — it's built for different use cases.

What about throttle for safety?

Some riders like throttle to quickly accelerate away from dangerous situations. Others never use it. It's a preference.
